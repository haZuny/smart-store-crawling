# Update Logs
- ***verseion 2.1***
  - 쇼핑몰의 주소를 입력하면, 해당 쇼핑몰의 상품 리스트를 저장하도록 수정(네이버 스마트스토어 한정)
  - 최대 탐색 페이지 설정 가능
  - 몇개씩 보기 설정 가능
  - 상품 정렬 기준 설정 가능
  - 엑셀에 각 상품번 순서 함께 표기
  - 엑셀에 상품의 대표 이미지 함께 표기
  - 엑셀의 셀 크기 자동 조절 추가
